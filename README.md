# napari-cilia-assistant tabbed analysis update

This update adds a five-step UI with Step 3 analysis tabs:

1. ROI Frequency
2. CBF Heatmap
3. Motion Activity
4. Advanced Flow

Copy the `src/napari_cilia_assistant/` folder into your repository, then run:

```bash
pip install -e .
napari
```

The UI intentionally uses Step 1 to Step 5 only. The previous `Step 6. Log` label is corrected to `Step 5. Export & Log`.
